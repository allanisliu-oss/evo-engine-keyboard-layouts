# EVO Engine 旗舰级“零位移”客制化生产力工具 - 产品定义 (PRD V2.0)

### 一、 产品核心定义 (Product Identity)

- **定位：** 面向文字工作者、程序员及重度数字创作者的旗舰级“零位移”客制化生产力工具。
    
- **形态：** 极简无缝的“上塑下铝”混合结构。外壳采用**象牙白（Ivory White）**温润色调，边缘细节均经过**圆润大 R 角处理**。机身整体追求**极致轻薄且纯平（0° 原生坡度）**，仅在一体式触控板与腕托接触面预留微小的舒适斜面，以缓解腕部压力。
    
- **核心理念：** 采用左右分区正交布局（Split ergonomic），通过中央数字导航矩阵、虚拟 TextStrip 与超大 TrackPad 的结合，配合苹果级多点触控与 Haptics 震动反馈，彻底消除右手在键盘与鼠标间的频繁位移。

### 二、 硬件与堆叠架构 (Hardware Architecture)

- **外壳材质 (混合型温润抗扰方案)：**
    
    - **底壳：** 6063 航空级铝合金 CNC 一体成型，表面 150 目以上阳极氧化/电泳。提供极致刚性与桌面稳定性。
        
    - **上盖与一体式手托：** CNC 加工的 **象牙白 POM（聚甲醛）** 或喷砂 PC（聚碳酸酯）。通过精细的曲面过渡，在纯平的机身之上为腕托区雕琢出微小的斜面坡度，提供天然温润、不冰手的亲肤触感，同时确保无线信号的完美穿透。

        
- **内胆结构 (独立悬挂 Top Mount)：**
    
    - **左右独立内胆：** 左手与右手区采用完全独立的 PCB 与定位板，释放中间区域给触控板和电池。
        
    - **固定方式：** 左右定位板独立锁附于 POM/PC 上盖（Top Mount），提供均匀、硬朗且干脆的敲击反馈，确保左右手感 100% 一致。
        
- **触控板“安全舱”物理隔离：**
    
    - 超大 TrackPad 不与上盖内胆接触，而是直接锁附于铝合金底壳 CNC 预留的刚性台阶上。彻底隔离键盘敲击的共振，确保触控按压坚如磐石。
        
- **轴体与键帽：**
    
    - **机械轴：** 凯华「鲸落 Mini」静音矮轴（2.8mm 行程，1.2mm 触发），兼顾机械段落感与极低噪音。
        
    - **键帽：** 现成 PBT 材质 DSA 高度（平胸球帽）。19mm 标准键距，无高度差完美契合分区正交矩阵。
        
- **底部与供电系统：**
    
    - 两块定制超薄聚合物锂电池分布于触控板两侧的底壳空腔内，并增加极薄气凝胶/泡棉隔热层，避免冬季低温影响电池活性。
        

### 三、 零位移交互与防误触中枢 (Interaction & Anti-Mistouch)

- **中央交互矩阵：** 整合运算符号、数字 0-9、Backspace/Delete 及方向键，形成直线型编辑流。
    
- **TrackPad 与 TextStrip：**
    
    - **硬件：** 约 120.00 x 75.00 mm 大尺寸电容触控板（参考 GlidePoint 方案），表面微低于两侧 POM/PC 手托 0.5mm-1.0mm 形成物理防误触屏障。
        
    - **虚拟分区逻辑 (固件级拦截)：** 顶部 15mm 区域被固件划分为 TextStrip。该区域屏蔽鼠标光标移动，仅提取 X 轴数据转换为横向滚动（用于视频时间轴拖拽/代码横移）。
        
- **语义化震动反馈 (Haptics)：**
    
    - 内置 3 颗 LRA 线性马达，配合 TI DRV2605L 等专用驱动芯片。
        
    - **刚性耦合：** 马达通过沉孔直接刚性锁附于铝合金底壳的触控板支撑台阶上，提供极其干脆的 Mac 级“咔哒 (Tick)”触感，拒绝廉价共振。
        

### 四、 核心电子架构 (Dual-MCU Architecture)

_由于大尺寸 TrackPad 需处理 120Hz 高频数据、防掌触运算并驱动无延迟震动反馈，彻底摒弃单主控方案，采用双轨制协处理器架构。为保证跨平台（Win/Mac）的极致稳定性与开发可行性，**通信协议定调为标准 HID Precision Touchpad (PTP)**。_

- **感知中枢 (协处理器，如 RP2040)：**
    
    - 专职负责通过 I2C 读取 TrackPad 原始绝对坐标数据。
        
    - 负责边缘计算：切割顶部 15mm 虚拟 TextStrip 区域、执行毫秒级的大面积掌压屏蔽（防误触）、计算接触面积阈值以模拟“重按”反馈。
        
    - 实时（<15ms 延迟）触发底部的 LRA 马达波形，提供点击的物理反馈感。
        
- **通信主控 (如 nRF52840)：**
    
    - 专职负责接收 RP2040 处理后的纯净 PTP 标准信号及键盘矩阵数据，并通过低功耗蓝牙 (BLE 5.0) 或 Type-C 极速发送至主机端。
        
- **系统生态与兼容性取舍 (The PTP Trade-off)：** 
    - 键盘全面拥抱标准 HID 协议，在 Windows 下可即插即用唤醒原生精确式触控板的全局手势。
    - 在 macOS 下，接受标准驱动下宏观切屏动画的极简物理特性（非无缝跟手跟随），以换取零逆向破解风险、极高的系统升级抗压能力及快速量产可能。
    

### 五、 磁吸保护套系统 (Magnetic Folio System)

- **功能：** 兼顾日常防尘、便携保护以及提供符合人体工学的打字倾角。
    
- **材质与结构：** 表面采用高品质环保仿皮材质（PU Leather），内部骨架夹层采用高密度再生 PET 板 (rPET) 或无卤素 FR4 板（低成本、高强度、合规）。
    
- **交互对位：** 键盘底壳与保护套内置 N52 钕磁铁阵列，实现无卡扣的高级“啪嗒”盲吸附体验。
    

### 六、 项目推进与客制化开团策略 (GTM Strategy)

- **原型机验证 (EVT Phase)：**
    
    - 使用 MJF 尼龙/树脂打印上盖 + 铝合金底壳，打样双 MCU 架构的第一版 PCB。重心在于**调教 PTP 协议的极低延迟响应、固化虚拟 TextStrip 的识别边界，以及对齐 Haptics 震动时序**。

        
- **视觉呈现与 IC 预热 (Content & Interest Check)：**
    
    - 制作高品质评测级视频（包含高帧率俯拍与屏幕 PiP 画中画），直观展示“零位移”带来的惊人效率提升、TextStrip 的工作原理以及 Haptics 反馈的沉浸感。
        
    - 全平台分发（YouTube, B站, zFrontier, Reddit 的 r/MechanicalKeyboards）。
        
- **小规模量产与众筹 (Group Buy / Crowdfunding)：**
    
    - 根据 IC 反馈微调配列与配色，确定最终 CNC 图纸。
        
    - 开启客制化圈定金 GB 积累第一波硬核口碑，后续可凭成熟的原型机与皮套系统直接登陆 Kickstarter 进行降维打击级的百万级众筹。
