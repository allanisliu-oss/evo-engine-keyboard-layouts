# EVO Engine 旗舰级“零位移”客制化生产力工具 - 产品定义 (PRD V2.0)

### 一、 产品核心定义 (Product Identity)

- **定位：** 面向**从未使用过机械键盘的 MacBook 用户、轻薄本用户及普通办公族**，提供无学习成本的桌面端高品质跨平台输入体验。
    
- **形态：** 极简无缝的“上塑下铝”混合结构。外壳采用**象牙白（Ivory White）**温润色调，边缘细节均经过**圆润大 R 角处理**。机身整体追求**极致轻薄且纯平（0° 原生坡度）**，仅在一体式触控板与腕托接触面预留微小的舒适斜面，以缓解腕部压力。
    
- **核心理念：** 采用左右分区正交布局（Split ergonomic），通过中央数字导航矩阵与**原生的超大 Precision Touchpad** 的结合，彻底消除右手在键盘与鼠标间的频繁位移。

### 二、 硬件与堆叠架构 (Hardware Architecture)

- **外壳材质 (混合型温润抗扰方案)：**
    
    - **底壳：** 6063 航空级铝合金 CNC 一体成型，表面 150 目以上阳极氧化/电泳。提供极致刚性与桌面稳定性。
        
    - **上盖与一体式手托：** CNC 加工的 **象牙白 POM（聚甲醛）** 或喷砂 PC（聚碳酸酯）。通过精细的曲面过渡，在纯平的机身之上为腕托区雕琢出微小的斜面坡度，提供天然温润、不冰手的亲肤触感，同时确保无线信号的完美穿透。

        
- **内胆结构 (独立悬挂 Top Mount)：**
    
    - **左右独立内胆：** 左手与右手区采用完全独立的 PCB 与定位板，释放中间区域给触控板和电池。
        
    - **固定方式：** 左右定位板独立锁附于 POM/PC 上盖（Top Mount），提供均匀、硬朗且干脆的敲击反馈，确保左右手感 100% 一致。
        
- **触控板“安全舱”物理隔离：**
    
    - 超大 TrackPad 不与上盖内胆接触，而是直接锁附于铝合金底壳 CNC 预留的刚性台阶上。彻底隔离键盘敲击的共振，确保触控按压坚如磐石。
        
- **轴体与键帽：**
    
    - **机械轴：** 凯华「鲸落 Mini」静音矮轴（2.8mm 行程，1.2mm 触发），兼顾机械段落感与极低噪音。
        
    - **键帽：** 现成 PBT 材质 DSA 高度（平胸球帽）。19mm 标准键距，无高度差完美契合分区正交矩阵。
        
- **底部与供电系统：**
    
    - 两块定制超薄聚合物锂电池分布于触控板两侧的底壳空腔内，并增加极薄气凝胶/泡棉隔热层，避免冬季低温影响电池活性。
        

### 三、 零位移交互中枢 (Interaction Center)

- **中央交互矩阵：** 整合运算符号、数字 0-9、Backspace/Delete 及方向键，形成直线型编辑流。
    
- **超大标准 TrackPad：**
    
    - **硬件：** 约 120.00 x 75.00 mm 大尺寸电容触控板（参考 GlidePoint / Azoteq PXM0091 方案），表面微低于两侧 POM/PC 手托 0.5mm-1.0mm 形成物理防误触屏障。
        
    - **原生体验：** 全面拥抱标准 HID (Windows Precision Touchpad, PTP) 协议。在 Windows 下即插即用唤醒全局多指手势；在 macOS 下由系统原生驱动提供稳定、低延迟的光标与轻扫控制。放弃复杂的虚拟分区，回归最纯粹的大面积手势交互。

### 四、 核心电子架构 (Single-MCU Architecture)

_由于剥离了复杂的马达震动和虚拟掌压切片算法，系统架构回归极简与极致稳定，大幅降低开发周期与量产风险。_

- **通信与控制主控 (如 nRF52840 或等效低功耗蓝牙 SoC)：**
    
    - 专职负责扫描非正交的键位矩阵信号。
    - 通过标准 I2C/SPI 总线直接读取 TrackPad 芯片的坐标点和手势事件，并打包为标准 PTP 协议。
    - 通过低功耗蓝牙 (BLE 5.0) 或 Type-C 极速发送至主机端。
        
- **固件生态：** 采用成熟的 ZMK 固件（主分支结合 `zmk-pointers` 分支），实现全平台的无线切屏与指针映射。
    

### 五、 独立配件体系：磁吸保护套 (Magnetic Folio Accessory)

- **产品定位：** 作为**可选独立配件 (Add-on Accessory)** 发售，裸机保持 0° 纯平（像携带一台 iPad）以满足极简外出需求，而磁吸套为桌面端重度办公提供人体工学倾角。
    
- **材质与结构：** 表面采用高品质环保仿皮材质（PU Leather），内部骨架夹层采用高密度再生 PET 板 (rPET) 或无卤素 FR4 板。
    
- **交互对位：** 键盘底壳与保护套内置 N52 钕磁铁阵列，实现无卡扣的高级“啪嗒”盲吸附体验。
    

### 六、 量产推进与降维打击策略 (Mass Market GTM Strategy)

- **受众转换 (Pivot to Broad Consumers)：**
    - 营销语言彻底剥离机械键盘客制化圈（键圈）的“打字音”、“Gasket 软弹”等圈地自萌属性。
    - 将整体产品定位为**“高端桌面消费电子”和“果粉的终极补完计划”**，主打一体化的高级感、跨平台的丝滑触控和出厂即巅峰的完整体验。

- **原型机验证 (EVT Phase)：**
    
    - 使用 MJF 尼龙/树脂打印上盖 + 铝合金底壳，打样极简的单 MCU 第一版 PCB。重心在于**调教 PTP 协议的极低延迟坐标上报与按键矩阵的稳定并发**。
        
- **视觉呈现预热 (Content Check)：**
    
    - 制作高品质苹果风商业广告级视频（包含高帧率俯拍与屏幕 PiP 画中画），直观展示“自带超大触控板的机械键盘”如何颠覆普通小白用户和设计师的工作流。
    - 分发渠道侧重于：B站数码区头部UP主、小红书高颜值桌面分享、YouTube 生产力科普向频道。
        
- **Kickstarter/众筹 (Crowdfunding Launch)：**
    
    - 避开键圈小团体的定金团（GB）模式带来的高期望值管理陷阱和发货周期争议。
    - 凭成熟的原型机与皮套配件体系，直接登陆 Kickstarter/Indiegogo 这种泛科技爱好者聚集地，进行降维打击级的百万美金级众筹。
